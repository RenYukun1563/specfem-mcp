# SPECFEM MCP Servers

[![arXiv](https://img.shields.io/badge/arXiv-2512.14429-b31b1b.svg)](https://arxiv.org/abs/2512.14429)
[![License](https://img.shields.io/badge/License-GPLv3-blue.svg)](LICENSE)

Model Context Protocol (MCP) server suite for SPECFEM family software (SPECFEM2D, SPECFEM3D, SPECFEM3D_GLOBE), enabling AI agents to perform seismic wave simulations through natural language interaction.

## Publication

**Seismology modeling agent: A smart assistant for geophysical researchers**

Yukun Ren, Siwei Yu, Kai Chen, Jianwei Ma
